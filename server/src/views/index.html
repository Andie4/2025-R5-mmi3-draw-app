Bienvenue sur l’API

Cette API permet d’accéder aux données de l’application. Vous pouvez utiliser les endpoints listés ci-dessous pour récupérer les informations dont vous avez besoin.

  • myUser:join
    – Un utilisateur cherche à rejoindre le dessin
  • myUser:joined
    – L'utilisateur a bien été crée, le serveur retourne l'utilisateur crée
  • user:joined
    – Un utilisateur vient de rejoindre le serveur
  • users:updated
    – La liste des utilisateurs vient d'être modifiée (edit d'un user, nouveau user, user qui quitte le serveur...)
  • user:edit
    – Pour permettre de modifier les informations d'un joueur (pseudo, avatar..)
  • user:left
    – Un joueur a quitté la partie
  • draw:start
    – Un utilisateur commence à dessiner
  • draw:move
    – Un utilisateur continue de dessiner
  • draw:end
    – L'utilisateur a fini de dessiner
  • draw:start
    – Un utilisateur commence à dessiner
  • draw:move
    – Un utilisateur continue de dessiner
  • draw:end
    – L'utilisateur a fini de dessiner
  • draw:clear
    – L'utilisateur a fini de dessiner
